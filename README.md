# Pastel Puzzle

A beautiful 3x3 sliding puzzle game with pastel blue and yellow theming.

**Features:**
- 3x3 sliding puzzle
- Move counter & timer
- Shuffle and reset buttons
- Win detection with celebration modal
- Mobile responsive

**To play:**
1. Click "Shuffle" to randomize the puzzle
2. Click tiles adjacent to the empty space to slide them
3. Arrange all tiles in order to win

**Colors:**
- Pastel Blue: #B4E7FF
- Pastel Yellow: #FFF9C4

Replace `puzzle.jpg` with your own 300x300px image for a custom puzzle.
